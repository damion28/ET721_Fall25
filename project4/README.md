# Student Learning Management App

## Description
The Student Learning App is a Django-based web application designed to enhance students' productivity and learning experience. Users can manage tasks, write blogs, and upload notes.

---

## Features
- **To-Do List:** Create, update, delete, categorize tasks, and mark as completed.
- **Blog Section:** Write blogs, add comments, view posts.
- **Notes Upload:** Upload and organize handwritten or digital notes with image preview.
